## 그리디 알고리즘

### 핵심이론

- `현재 상태에서 선택지 중 최선의 선택지`가 `전체 선택지 중 최적의 선택지`라고 생각하는 방법
  - 하지만 항상 최적의 해를 보장하지 않는다.

1. 현재 상태에서 선택지 중 최선의 선택지 선택
2. 현재 선택한 해가 적절한지 검사한다.
3. 지금까지 선택한 해 집합이 문제를 해결할 수 있는지 검사한다.
   - 아니라면 1번으로 가서 반복한다.
